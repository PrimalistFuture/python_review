API Endpoints and API Parameters

APIs are a set of commands, functions, protocols, and objects that programmers can use to create software or interact with external systems.

Essentially APIs are barriers between your program and some external software or system.

Often used to get some other entities data.

The endpoint is some usually some URL that you can get some data from.

A request has to be made over the internet. It can be as simple as going to the endpoint, but it usually wants you to format the request in a certain way and give them certain data about your credentials or whatever.

API can also have parameters, which can be used to configure the data that you get back

Good API documentation will tell you want parameters the API expects, and which are optional, as well as what shape they want the parameter to be.
