Absolute vs Relative Path

A single forward / indicates your root folder. An absolute file path always starts are the root. But a relative path starts relative to the folder we are currently in. Check the demo for how to figure out the folder you are currently working in!

A . indicates to look at the current folder ./Reports/Weeks. But the ./ is optional for files within the current Reports/Weeks directory

Two .. indicates to look up ../report.doc
